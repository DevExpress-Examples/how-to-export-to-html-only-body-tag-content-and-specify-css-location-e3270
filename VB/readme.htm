Export to HTML - How to export only tag content and specify the location of CSS styles

This example illustrates how you can use the HtmlExporter class to export the document in HTML format. The Export method enables you to emit only content encompassed by the specified root tag. The HtmlDocumentExporterOptions object passed as the method's parameter specifies the location of CSS styles - they can be exported to external file, located within the style tag or embedded as inline styles in the corresponding HTML tags. The code uses a custom UriProvider to accomplish this task. It is instantiated and registered via the RegisterProvider method of RichEditControl's IUriProviderService service.
